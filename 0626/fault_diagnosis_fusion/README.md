# 工业设备故障诊断多传感器融合（大作业题目 8）

基于 **振动 + 声学 + 红外温度** 三类传感器的特征级融合，对滚动轴承四种工况
（正常 / 内圈故障 / 外圈故障 / 滚动体故障）进行故障诊断，并对比单传感器与融合的识别准确率。

> 无真实传感器，数据按物理机理 **mock 生成**（见 `src/mock_sensors.py`）。

## 环境

```bash
pip install -r requirements.txt
```

依赖：numpy、scipy、matplotlib、scikit-learn。中文绘图需系统含「微软雅黑 / 黑体」字体（Windows 自带）。

## 运行

```bash
# 端到端：生成数据 → 提取特征 → 融合实验 → 出图 → 保存指标
python src/pipeline.py

# 交互式演示（录演示视频用）：点击按钮随机抽样并实时诊断
python demo/demo.py
```

产物：
- `figures/`：6 张图（波形、频谱、声学/温度、准确率对比、混淆矩阵、特征重要性）
- `results/metrics.json`、`results/classification_reports.txt`：量化指标

## 方法

| 环节 | 说明 |
|---|---|
| 数据 | 4 工况 × 200 样本；振动/声学 12 kHz×2048 点，温度 60 点序列 |
| 特征 | 振动时域(RMS/峭度/峰值因子…) + 频域(BPFI/BPFO/BSF 频带能量) + 声学(频谱熵/高频占比/异响频带) + 温度(均值/最大/温升梯度/热点) 共 18 维 |
| 融合 | 特征级融合（三传感器特征拼接）→ 随机森林分类 |
| 对比 | 仅振动 / 仅声学 / 仅温度 / 三传感器融合 |

## 结果（seed=42）

| 配置 | 准确率 |
|---|---|
| 仅振动 | 88.3% |
| 仅声学 | 63.3% |
| 仅温度 | 59.6% |
| **三传感器融合** | **94.2%** |

融合较最佳单传感器提升约 **5.8 个百分点**：振动擅长区分故障类型、声学擅长判断有无异响、
温度反映严重程度，三者信息互补，故融合显著优于任一单传感器。
