📖 骨架书

《Pro Git》 — Scott Chacon、Ben Straub(官方免费书)

Git 入门到精通的官方权威读物。它的脉络是:先讲清 Git 是什么——它把数据当作一系列快照而非差异, 理解了这一点其余概念都顺理成章;再带你跑通基础日常操作(暂存、提交、查看历史、撤销); 接着重点剖析 分支模型——这是 Git 的杀手锏,创建分支几乎零成本,合并与变基让协作如丝般顺滑; 然后进入远程协作(fetch/pull/push、Pull Request、Fork);最后深入重写历史与底层原理 (reset/reflog、Git 对象模型)。下面的 Tab 即对应书的核心章节。
