outline 与 box-shadow

两个画在盒子「外面」的效果。一个关乎可访问性，一个关乎视觉层次——共同点是：它们都不占布局空间。

一、outline：不占空间的轮廓

outline（轮廓）看起来很像 border，但有一个本质区别：outline 画在 border 外侧，且完全不占据布局空间、不影响盒子尺寸。它像是「浮」在盒子上方的一圈描边，相邻元素根本感知不到它的存在。

对比borderoutline
占布局空间占（撑大盒子 / 触发重排）不占（叠加绘制）
位置盒子边界border 外侧
四边能否分开设能（border-top 等）不能，永远整圈
能否圆角能（border-radius）现代浏览器跟随元素圆角
outline-offset无有，可让轮廓离开元素一段距离
outline 紧贴 outline-offset: 4px

两个盒子的实际占位完全相同——outline 没把任何一个撑大。

二、为什么 outline 对可访问性至关重要

浏览器默认会给获得焦点的元素（点击或 Tab 键导航到的按钮、链接、输入框）加一圈 outline，这就是焦点指示器（focus ring）。它告诉键盘用户「现在选中的是这一个」。

反例（极其常见的错误）：*:focus { outline: none; } 或 button:focus { outline: 0; }。设计师嫌焦点框「难看」，一刀切全去掉。

问题：这让所有只能用键盘操作的用户（视障用户、运动障碍用户、临时不能用鼠标的人）彻底迷失——他们按 Tab 在页面上跳，却完全看不到光标在哪。这是 WCAG（无障碍标准）明确的违规项，也是大厂前端面试的高频「红线」考点。

正例 · 不要去掉，要替换成更好看的
/* 用 :focus-visible 只在键盘导航时显示，鼠标点击不显示 */
button:focus-visible {
  outline: 2px solid #60a5fa;
  outline-offset: 2px;
}
/* 想完全自定义？用 box-shadow 模拟焦点环也行，但务必保留可见反馈 */
:focus-visible 是现代解法：浏览器智能判断——用键盘 Tab 进来才显示焦点环，用鼠标点击则不显示，既美观又不丢可访问性。永远不要无脑 outline: none。

三、box-shadow 语法

阴影给盒子带来「层次感」和「立体感」，是卡片、弹窗、悬浮按钮的灵魂。语法有六个部分：

CSS
box-shadow: offset-x offset-y blur spread color inset;
/*           水平偏移  垂直偏移  模糊  扩展   颜色   内阴影 */

.card {
  box-shadow: 0 4px 12px 0 rgba(0,0,0,.3);
  /* 不偏移X, 下移4px, 模糊12px, 不扩展, 半透明黑 */
}
参数作用
offset-x / y阴影相对元素的位移。正 y 向下，正 x 向右
blur模糊半径，越大越柔和虚化，0 是硬边
spread扩展半径，正值让阴影整体变大，负值缩小
color阴影颜色，常用半透明 rgba 才自然
inset加上它阴影变成内阴影（凹陷效果）
投影卡片
假边框
inset 内阴影
光晕 glow

四、box-shadow 的妙用

  • 卡片悬浮：0 4px 12px rgba(0,0,0,.15)，hover 时加大阴影制造「浮起」动效。
  • 当假边框：0 0 0 2px color（无偏移无模糊、只用 spread）= 一圈不占空间的「边框」，且能配合圆角、能做焦点环替代品。
  • 多重阴影：用逗号叠加多层，做精致的 Neumorphism 或多方向投影。
CSS · 多重阴影（逗号分隔，先写的在上层）
.fancy {
  box-shadow:
    0 1px 2px rgba(0,0,0,.2),   /* 近处硬投影 */
    0 8px 24px rgba(0,0,0,.3);  /* 远处柔投影 */
}

五、box-shadow vs filter: drop-shadow

还有一个画阴影的方式 filter: drop-shadow()，两者经常被搞混：

box-shadowfilter: drop-shadow()
跟随的形状盒子的矩形边界（含 border-radius）元素的实际不透明像素轮廓
对 PNG 透明图给整个矩形加影（包括透明区）只给图案本身加影（沿轮廓）
支持 spread / inset支持不支持
性能较好filter 走合成层，可能更重
选择口诀：给规整的矩形/圆角卡片加影，用 box-shadow。给不规则形状（透明 PNG 图标、SVG、用 clip-path 切出的异形）加贴合轮廓的影，用 filter: drop-shadow()——它能沿着图案边缘画影，而 box-shadow 只会给那个看不见的矩形外框画影。

六、共同点回顾

本节两个主角都画在盒子「外面」、都不占布局空间、不触发重排（reflow）——它们只在绘制（paint/composite）阶段生效。这也是为什么用 box-shadow 做 hover 边框比改 border 更平滑：改 border 会改变盒子尺寸引发重排和布局抖动，而 box-shadow 不会。

小结

  • outline：画在 border 外、不占空间、整圈不可分边；是焦点指示器，永远不要无脑 outline:none，要用 :focus-visible 替换。
  • box-shadow：六参数 x y blur spread color inset，可多重叠加，可做投影/假边框/光晕/内阴影。
  • 异形/透明图的贴合阴影用 filter: drop-shadow()，规整矩形用 box-shadow。
  • 两者都不占布局、不触发重排，做动效比改 border 更顺滑。
