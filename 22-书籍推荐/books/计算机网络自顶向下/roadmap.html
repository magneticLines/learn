📖 Computer Networking: A Top-Down Approach, 8th Ed.

最好的计算机网络入门教材

作者: James F. Kurose & Keith W. Ross

适合谁:计算机专业学生/自学者。前置知识:会写简单程序即可（书中的实验用 Python/Java）。

阅读建议:每一章读完后一定要做 Wireshark 实验。第 2-4 章是最核心的部分（应用层 → 传输层 → 网络层）。第 5 章（链路层）可略读，第 6 章（无线网络）按需。第 7-8 章（多媒体/安全）偏概览。

避坑:这本书不是 RFC 手册——不需要记每个协议头部的每一位，重点是理解"为什么这么设计"。

  • 第 1 章 · 计算机网络总览
    • 互联网的构成:边缘 / 核心 / 接入网
    • 分组交换 vs 电路交换
    • 时延构成:处理 / 排队 / 传输 / 传播
    • 协议分层与 OSI / TCP/IP 模型核心
  • 第 2 章 · 应用层
    • 应用层架构:CS / P2P
    • HTTP:持久连接 / Cookie / 缓存 / 条件 GET核心
    • FTP:控制连接与数据连接分离
    • DNS:层次命名空间 / 递归迭代查询 / DNS 缓存核心
    • Socket 编程:UDP / TCP Socket 实验
  • 第 3 章 · 传输层
    • 多路复用与多路分解
    • UDP:无连接 / 校验和
    • 可靠数据传输原理:rdt 协议家族核心
    • TCP:头部 / 序号确认 / 流量控制 / 连接管理核心
    • TCP 拥塞控制:AIMD / 慢启动 / 快速重传 / 快速恢复核心高频面试
  • 第 4 章 · 网络层:数据平面
    • 虚电路 vs 数据报网络
    • IP 协议:IPv4 头部 / 分片 / 编址核心
    • NAT 与 DHCP核心
    • IPv6:动机 / 头部对比 / 过渡
  • 第 5 章 · 网络层:控制平面
    • 路由算法:Dijkstra / 距离向量 / RIP / OSPF
    • BGP:路径向量 / 策略路由进阶
    • SDN 与控制平面分离进阶
  • 第 6-7 章 · 链路层与安全(概要)
    • ARP:地址解析协议
    • 以太网与交换机:自学习 / VLAN
    • HTTPS / TLS 握手呼应第 8 章
