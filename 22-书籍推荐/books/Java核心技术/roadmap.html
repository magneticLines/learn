📖 Core Java Volume I — Fundamentals, 12th Ed.

Java 入门第一书

作者: Cay S. Horstmann

适合谁:零基础或有一点编程经验想系统学 Java 的人。前置知识:基本的编程概念（变量、循环、函数）。

阅读建议:第 1-7 章是必读基础；第 8-9 章（泛型、集合）是 Java 日常开发核心；第 12 章（并发）与《Java 并发编程实战》配合读效果更好。第 10-11 章（Swing）可跳过，不影响主线。

  • 第 1-3 章 · Java 入门
    • Java 的设计哲学与生态一次编写到处运行 / JVM / 垃圾回收
    • 基本语法:变量、运算符、控制流
    • 字符串与 StringBuilder
    • 输入输出基础:Scanner、格式化输出
  • 第 4-5 章 · 面向对象
    • 类、对象、封装核心
    • 构造器、static、方法参数传递
    • 继承、多态、super核心
    • Object 类:equals、hashCode、toString
    • 抽象类、接口、sealed 类Java 17+
  • 第 6-7 章 · 进阶语言特性
    • lambda 表达式与函数式接口核心
    • 内部类与匿名类
    • 异常体系与 try-with-resources核心
    • 日志与断言
  • 第 8-9 章 · 泛型与集合
    • 泛型:类型擦除、桥接方法、通配符核心高频面试
    • 集合框架总览:Collection / Map 体系
    • ArrayList vs LinkedList、HashMap vs TreeMap
    • HashMap 的实现原理核心高频面试
    • Collections 工具类与不可变集合
  • 第 12 章 · 并发
    • 线程基础:Thread / Runnable / 线程状态
    • synchronized 与 volatile 基础
    • 阻塞队列与并发集合
    • 线程池与 Callable / Future
    • CompletableFuture 入门
