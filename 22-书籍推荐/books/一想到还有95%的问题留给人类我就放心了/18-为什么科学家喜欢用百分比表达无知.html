为什么科学家喜欢用百分比表达无知

"不知道"是个含糊的词，"95% 不知道"却是一把尺子——科学家把无知量化，恰恰是为了正视它。

书名不是修辞：那 95% 是算出来的

整本书的书名——"一想到还有 95% 的问题留给人类"——听起来像句俏皮话，其实背后是一个实打实的估算。把可观测宇宙的全部物质-能量总账摊开，你会发现我们真正"认识"的那部分，少得可怜。

5% 暗物质 27% 暗能量 68%
成分占比我们知道什么
普通物质
原子、恒星、行星、你		约 5%了解得相当透彻，是标准模型的地盘
暗物质约 27%知道它有引力，但不知道它是哪种粒子
暗能量约 68%知道它在推动宇宙加速膨胀，本质几乎全是谜

把后两行加起来就是那个著名的数字：

27% + 68% = 95% 的宇宙，我们叫不出名字
所以"95%"是字面意义上的:它不是"很多问题"的夸张说法，而是"宇宙物质-能量构成里，约 95% 是我们完全不了解的"这一事实的直接换算。书名把一个宇宙学结论，写成了一句让人安心的玩笑。

把"不知道"变成一把尺子

科学家钟爱百分比，本质是一种量化未知的策略。同样是承认无知，"我不太懂"和"我对其中 95% 不懂"是两种完全不同的表述——后者是可以被追问、被检验、被逐年更新的。

① 从"模糊的不知道"到"具体的 95%"

说"宇宙很神秘"等于什么都没说。但说"95% 的物质-能量构成是未知的"，你立刻就知道这个洞有多大、有多严重。百分比给无知装上了刻度，让它变成一个可以下手的问题，而不是一团情绪。

② 百分比本身就是一种进步

能报出"95%"这个数，前提是你已经把已知的 5% 摸清、并且察觉到了剩下那块的存在。半个多世纪前，人类甚至不知道暗物质、暗能量的存在——那时的"未知"是 100%、且毫无察觉。

认知的进化：无知也分段位 第 0 层 "宇宙就是原子构成的" <- 不知道自己不知道（最危险） │ 第 1 层 "咦，星系转得不对劲……" <- 察觉到有东西没算进去 │ 第 2 层 "未知约占 95%，分暗物质/暗能量" <- 能把无知量化 & 分类 │ 第 3 层 ？直接探测到它是什么？ <- 还没到，留给人类的部分

换句话说，"我们知道自己不知道"本身就是一次认知升级。能写出 95% 这个百分比，是站在第 2 层往上看，而不是还蒙在第 0 层。

③ 给过度自信踩一脚刹车

百分比还是对抗 false certainty（虚假的确定感）的解毒剂。同一句话，加不加那个百分比，诚实程度天差地别：

听起来很笃定
"宇宙是由原子组成的。"

（听众以为：搞定了，
 我们懂宇宙了。）
科学的诚实
"宇宙里的原子
 只占约 5%。"

（听众意识到：哦，
 大头还没搞懂。）
常见误区:把"科学已经解释了 X"当成"X 已经完全没有未知"。一个负责任的百分比，恰恰是在提醒你——已知的边界到哪儿为止，别把 5% 的确定当成 100% 的真相。
关键结论:科学家用百分比表达无知，不是为了显得谦虚，而是因为百分比同时做到了三件事——量化问题的大小、标记认知的进度、约束过度自信。无知一旦有了数字，就从"绝望"变成了"待办清单"。

无知不是弱点，是探索的理由

这也正是整本书的基调：面对 95% 的空白，作者的态度不是焦虑，而是"放心"——因为这意味着最精彩的问题都还没被做完，人类有的是活儿干。

补一刀:书里有段妙论——"如果你第一次约会时说'我几乎什么都不懂'，对方大概不会想见你第二次。但如果你说'在我最熟悉的领域，我对 95% 的东西还是不懂'——对方反倒会觉得你诚实又有趣。"无知说得太笼统像摆烂，量化之后就成了魅力。百分比，连搭讪都用得上。
