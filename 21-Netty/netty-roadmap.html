📖 骨架书

《Netty in Action》 — Norman Maurer、Marvin Allen Wolfthal

Netty 核心提交者亲自撰写,是理解 Netty 设计思想最权威的读物。它的脉络是:先解释为什么需要异步事件驱动—— 同步阻塞 IO 在高并发下线程资源撑不住,这是 Netty 存在的根本原因;再拆解 Channel / EventLoop / ChannelFuture 三大核心接口的分工;然后深入 ByteBuf 内存模型 与 ChannelHandler/Pipeline 职责链; 接着讲 编解码框架 如何把字节流变成业务对象(以及粘包/拆包这一 TCP 天生的问题); 最后落到 WebSocket / UDP 等具体协议实现 与案例。下面的 Tab 即对应书的核心章节。

🔧 配套开源项目

sanshengshui/netty-learning-example

GitHub 上最经典的 Netty 中文实践仓库之一(2.6k+ Star),按 netty-helloworld → netty-http → netty-websocket → 自定义协议 → 整合 SpringBoot/Kafka/Protobuf 的顺序组织了一系列可直接运行的示例。 书本讲透"为什么"这么设计,这个仓库负责补上"实际项目里长什么样"——每学完一个 Tab,建议去仓库里找对应模块跑一遍、改一改参数, 光看书上的代码片段是建立不起工程直觉的。
