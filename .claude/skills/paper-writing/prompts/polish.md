# Prompt — Academic Language Polishing

## Task
Improve the academic quality, clarity, and style of the provided text without changing its meaning.

## Instructions

1. **Grammar & syntax** — Fix grammatical errors, awkward phrasing, and run-on sentences.
2. **Academic register** — Replace informal or colloquial language with academic equivalents.
3. **Conciseness** — Remove redundancy while preserving meaning.
4. **Logical flow** — Improve transitions between sentences and paragraphs.
5. **Terminology consistency** — Ensure technical terms are used consistently throughout.
6. **Hedging** — Add or adjust hedging language where claims need qualification.

## Output Format

Provide a side-by-side diff when possible:

| Original | Polished |
|----------|----------|
| ... | ... |

Then provide the full polished text below the table.

## Chinese-specific rules (中文润色)
- 避免口语化表达，使用学术书面语
- 长句适当拆分，逻辑关系用连词明确表示
- 专业术语保持统一，首次出现时标注英文对照
