# 中国科学系列期刊模板

> 适用：《中国科学》各分册（信息科学、技术科学、生命科学、地球科学、物理学等）及《科学通报》

## 期刊分册

| 刊名 | 学科 | ISSN |
|------|------|------|
| 中国科学：信息科学 | 计算机、通信、自动化 | 1674-7267 |
| 中国科学：技术科学 | 机械、材料、能源、化工 | 1674-7321 |
| 中国科学：生命科学 | 生物、医学、农业 | 1674-7232 |
| 中国科学：地球科学 | 地质、大气、海洋 | 1674-7755 |
| 中国科学：物理学 力学 天文学 | 物理、力学、天文 | 1674-7348 |
| 科学通报 | 综合性，快报形式 | 0023-074X |

## 基本结构（研究论文）

```
题目（中英文）

作者¹,²，作者²
¹ 单位, 城市 邮编
² 单位, 城市 邮编
† 通信作者，E-mail: xxx@xxx.edu.cn
收稿日期：XXXX-XX-XX；接受日期：XXXX-XX-XX

摘要（200~300字）
关键词（5~8个，以分号分隔）

Abstract（与中文对应）
Keywords

1  引言

2  研究方法（或：理论分析）
   2.1  ……
   2.2  ……

3  结果
   3.1  ……

4  讨论

5  结论

致谢

参考文献
```

## 《科学通报》快报格式

```
题目

作者（单位）

正文（无需分节，一般 2000~4000 字，图表不超过 4 幅）

参考文献（不超过 20 条）
```

## 版面与格式要求

| 项目 | 中文稿 | 英文稿 |
|------|--------|--------|
| 正文字体 | 宋体小四 / TNR 12pt | Times New Roman 12pt |
| 行距 | 1.5 倍 | Double-spaced |
| 摘要 | ≤ 300 字 | ≤ 200 words |
| 关键词 | 5~8 个 | 5~8 terms |
| 图分辨率 | ≥ 600 dpi（线图）/ ≥ 300 dpi（照片） | 同左 |
| 参考文献 | 顺序编码制，GB/T 7714 | 同左 |

## 参考文献示例

```
期刊：
[1] 张三, 李四, 王五. 深度强化学习综述[J]. 中国科学: 信息科学, 2023, 53(4): 601-629. DOI: 10.1360/SSI-2022-0xxx

英文期刊：
[2] Smith A, Jones B. Title of the paper. Sci China Inf Sci, 2023, 66: 119101. DOI: 10.1007/s11432-022-xxxx-x

专著：
[3] 作者. 书名[M]. 出版地: 出版社, 年份. 引用页码

网络资源：
[4] 作者. 题名[EB/OL]. (发布日期)[引用日期]. URL
```
