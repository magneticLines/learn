# 高校社会科学学报模板（文史哲类）

> 适用：各高校学报（社会科学版）、CSSCI 来源期刊、北大核心（文史哲类）

## 基本结构

```
题目（中文，一般不超过 20 字，副标题用破折号引出）

作者（单位，省市 邮编）

摘要：……（150~300字，不宜过于结构化，多为段落式叙述）
关键词：词1；词2；词3；词4（3~5个）

[中图分类号] XXXX    [文献标识码] A    [文章编号] XXXX-XXXX(年)期-页-页数

一、引言（或：一、问题的提出）

二、……（正文按一、二、三……一级标题；（一）（二）二级标题；1. 2. 三级标题）

三、……

四、结语（或：结论与启示）

注释
① ……（随文脚注或文末尾注，非正式参考文献用注释）

参考文献
[1] 作者. 题名[J]. 刊名, 年份(期): 页码.
[2] 作者. 书名[M]. 出版地: 出版社, 年份.
```

## 与理工类学报的主要区别

| 项目 | 社科版 | 理工版 |
|------|--------|--------|
| 摘要风格 | 段落式，叙述性 | 结构化四要素 |
| 标题编号 | 一、（一）1. | 1  1.1  1.1.1 |
| 注释 | 脚注/尾注（文献外的说明性内容） | 一般不用注释 |
| 英文摘要 | 部分期刊要求 | 通常必须 |
| 图表 | 较少，以文字论述为主 | 图表为核心呈现方式 |

## 版面规范

| 项目 | 要求 |
|------|------|
| 正文字体 | 宋体，小四号（12pt） |
| 标题 | 一级标题黑体四号；二级标题黑体小四；三级标题仿宋小四 |
| 行距 | 1.5 倍行距 |
| 摘要/关键词 | 楷体，五号（10.5pt） |
| 注释 | 小五号，脚注形式 |

## 引用规范

社科类常见同时使用两套引用系统：

1. **参考文献**（文后，标注已直接引用的文献）
   > [1] 费孝通. 乡土中国[M]. 北京: 北京出版社, 2005: 12-15.

2. **注释**（脚注或尾注，用于补充说明、间接引用、引用古籍）
   > ① 此处"礼治"概念参见费孝通先生的相关论述，但本文有所延伸。

## CSSCI 收录期刊投稿建议

- 摘要应清晰呈现：问题意识、研究方法、核心论点、学术贡献
- 关键词应与中图分类号对应，便于数据库检索
- 参考文献以近 5 年文献为主，注意引用同刊文章（符合期刊学术范围）
