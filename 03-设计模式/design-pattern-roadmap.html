创建型模式 — 关注对象的创建过程，将对象的创建与使用分离

  • 单例模式
    • 饿汉式 / 懒汉式 / 双重检查锁 / 静态内部类 / 枚举
    • 线程安全问题与序列化破坏
  • 工厂方法模式
    • 简单工厂（非标准）
    • 工厂方法（定义一个创建对象的接口，让子类决定实例化哪个类）
    • 实际应用：Collection.iterator()
  • 抽象工厂模式
    • 产品族与产品等级结构
    • 与工厂方法模式的区别
    • 实际应用：java.sql.Connection
  • 建造者模式
    • 标准建造者模式
    • 链式调用变体
    • Lombok @Builder / OkHttp Request.Builder
  • 原型模式
    • 浅拷贝 vs 深拷贝
    • Cloneable 接口
    • 序列化实现深拷贝

结构型模式 — 关注类和对象的组合，形成更大的结构

  • 适配器模式
    • 类适配器 vs 对象适配器
    • Java I/O: InputStreamReader / OutputStreamWriter
    • Spring HandlerAdapter
  • 装饰器模式
    • 动态扩展功能 vs 继承
    • Java I/O: BufferedInputStream / ZipInputStream
    • Spring Transactional 嵌套事务
  • 代理模式
    • 静态代理 vs JDK 动态代理 vs CGLIB
    • Spring AOP 核心原理
    • @Transactional 自调用失效问题
  • 外观模式
    • 简化复杂子系统的一站式接口
    • Spring JdbcTemplate
  • 桥接模式
    • 抽象与实现分离
    • JDBC Driver 驱动设计
    • Spring 数据源抽象
  • 组合模式
    • 树形结构处理（部分-整体层次）
    • 文件系统示例
    • Spring CacheManager 组合
  • 享元模式
    • 共享细粒度对象
    • String 常量池 / Integer 缓存池
    • 线程池复用原理

行为型模式 — 关注对象之间的通信和职责分配

  • 策略模式
    • 算法族封装，消除 if-else
    • Comparator 策略模式
    • 结合工厂模式的最佳实践
  • 观察者模式
    • 发布-订阅机制
    • Spring Event / @EventListener
    • Guava EventBus
  • 模板方法模式
    • 算法骨架定义 + 钩子方法
    • Spring JdbcTemplate
    • Spring RestTemplate
  • 责任链模式
    • 请求链式处理
    • Java Filter / Spring Interceptor
    • Netty Pipeline
  • 迭代器模式
    • 遍历集合与 for-each 原理
    • Java Iterator / Iterable
  • 状态模式
    • 状态驱动行为 vs if-else 状态机
    • 与策略模式的区别
  • 命令模式
    • 请求封装为对象
    • 撤销 / 重做功能
    • Runnable 与 Thread 池
  • 备忘录模式
    • 状态保存与恢复
    • 快照功能实现
  • 中介者模式
    • 降低多对象间的耦合
    • MVC 中的 Controller
  • 解释器模式
    • 定义语法规则与表达式解析
    • 正则表达式引擎
  • 访问者模式
    • 双重分派机制
    • 分离数据结构与操作
  • Spring 框架中的设计模式
    • 单例模式 — Bean 默认作用域
    • 工厂模式 — BeanFactory / ApplicationContext
    • 代理模式 — AOP 动态代理
    • 模板方法 — JdbcTemplate / RestTemplate
    • 观察者模式 — ApplicationEvent / @EventListener
    • 策略模式 — ResourceLoader / InstantiationStrategy
    • 适配器模式 — HandlerAdapter / AdvisorAdapter
    • 责任链模式 — Interceptor / Filter 链
  • MyBatis 框架中的设计模式
    • 建造者模式 — SqlSessionFactoryBuilder
    • 工厂模式 — SqlSessionFactory
    • 代理模式 — Mapper 接口动态代理
    • 模板方法 — BaseExecutor
  • JDK 源码中的设计模式
    • 装饰器模式 — I/O 流体系
    • 迭代器模式 — Collection / Iterator
    • 策略模式 — Comparator
    • 观察者模式 — Observer / Observable
    • 享元模式 — String 常量池 / Integer 缓存
    • 单例模式 — Runtime 类
  • 其他框架
    • Netty — 责任链 Pipeline / 装饰器 ChannelHandler
    • Jackson — 策略模式 JsonParser / 访问者模式 JsonNode
    • SLF4J — 外观模式 Logger 门面
